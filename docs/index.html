Tania An open source farm management system and it's written in Go.

  • home
  • api reference
  • About

You can easily manage growing areas, reservoirs, farm tasks, inventories, and crops growing progress. Working on small backyard garden or even big farm getting more managable.

Main Features

Crops

Track your crops growth since the beginning
to harvesting.

Areas

Setup and manage unlimited areas for nurseries
and growing crops.

Reservoirs

Setup, attach, and manage unlimited reservoirs
to any areas.

Inventories

Record your farm's seeds supplies and track what area is using it.

Tasks

Manage all operations on your farm
and never forget it.

REST API

Build with REST API in the mind, you can build your client app or integrate Tania to your app.

Clone and build Tania for your operating system from our Github. You need to setup Go programming language and NPM (Node Package Manager).
